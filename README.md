# Tripleten web_project_around

Este projeto foi desenvolvido com HTML5, CSS3 e JavaScript ES6, utilizando a
metodologia BEM para organização dos estilos e uma arquitetura baseada em
componentes (Card, Section, Popup, PopupWithForm, PopupWithImage, FormValidator
e UserInfo). Foram aplicadas técnicas de Programação Orientada a Objetos,
validação de formulários usando a API nativa de validação do navegador,
manipulação dinâmica do DOM para criação e remoção de cards, além de pop-ups
acessíveis que podem ser abertos e fechados por cliques, teclado e sobreposição.
